# ДЗ:

Создать форму регистрации в дурдоме

Поля: 
- Имя
- Фамилия
- Отчество
- Номер телефона
- email
- Дата рождения (вспоминай как делается)
- галочки:
- - согласен с правилами нахождения в учереждении
  - я фрик
  - я не сбегу из дурдома честно

- далее 3 радиокнопки с выбором диагноза (на свое усмотрение типа шизофрения/раздвоение личности/....)
- кнопка отправить (НЕДОСТУПНА, ПОКА НЕ ВЫСТАВЛЕНЫ ВСЕ ГАЛОЧКИ, ПОКА ВСЕ ПОЛЯ НЕ БУДУТ ВАЛИДНЫМИ, ПОКА ХОТЯ БЫ ОДНА РАДИОКНОПКА НЕ БУДЕТ НАЖАТА)

Буду проверять пытаться вводить невалидные данные
Вся информация о пациенте должна выводиться в alert (типа поздравляем, ${имя}? добро пожаловать в психбольницу №24, тут рады людям с диагнозом ${диагноз}). Опять таки на твое усмотрение, главное чтобы вывод был забавный
