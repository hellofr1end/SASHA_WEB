# Разворот строки
### Суть: Пользователь вводит слово, жмет кнопку, и слово пишется задом наперед (например под полем input) в заголовке h1
 например: [
    привет: тевип,
    пока: акоп,
    кот: ток,
    да: ад,
    нет: тен
 ]
