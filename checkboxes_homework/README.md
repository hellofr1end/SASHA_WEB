# Разворот строки
### Суть: Пользователь вводит слово. При нажатии на чекбокс - пишется задом наперед (например под полем input) в заголовке h1. При отжатии - пишется обычно не задом наперед
 например: [
    привет: тевип,
    пока: акоп,
    кот: ток,
    да: ад,
    нет: тен
 ]
