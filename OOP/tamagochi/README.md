# Тамагочи

## Задача

Создай простое приложение "Тамагочи" с использованием класса и HTML.

---

## 1. Создай класс `Pet`

### Поля класса

- name — имя питомца
- health — здоровье (100)
- hunger — голод (0)
- energy — энергия (100)

Значения здоровья, голода и энергии должны задаваться в конструкторе.

---

## 2. Добавь методы

### `feed()`

- уменьшает голод

### `play()`

- увеличивает голод
- уменьшает энергию

### `sleep()`

- восстанавливает энергию

Не допускай значений меньше 0 и больше 100.

---

## 3. Создай объект

Создай одного питомца.

Например:

```js
let pet = new Pet("Барсик");
```

---

## 4. Сделай HTML

На странице должны отображаться:

- имя питомца
- здоровье
- голод
- энергия

Также должны быть три кнопки:

- Покормить
- Поиграть
- Уложить спать

---

## 5. Сделай функцию `render()`

Она должна обновлять информацию на странице, используя значения полей объекта `pet`.

После каждого изменения питомца необходимо вызывать `render()`.

---

## 6. Подключи кнопки

При нажатии на кнопку:

- "Покормить" вызывается `pet.feed()`
- "Поиграть" вызывается `pet.play()`
- "Уложить спать" вызывается `pet.sleep()`

После этого информация на странице должна обновляться.

Пользователь открывает страницу, видит характеристики питомца и может менять их с помощью кнопок.
